Reddit CopyCat API Documentation

Below there're all commands to retrieve information from some Reddit Posts stored on the Redis DB.

Learn more »

Technologies Used

Python 3.6 / Flask 1.1.1

Redis 5.0.5

Bootstrap 4.3.1

Allowed HTTPs requests

  • PUT    : Create a new "Reddit" Post
  • POST   : Update the Vote Count of a Post
  • GET    : Get a post or list of post
  • DELETE : To delete a Post
GET POST PUT DELETE

/posts Get all posts stored on DB

/r/<community>/<post_id> Get a post stored on DB, if it exists. (Example: /r/uady_community/post:968658748)

/create_new_post Actually, it's a built-in service that sends you a form to create a new post

/remove_vote/<post_id> Add a vote to a Post

/add_vote/<post_id> Remove a vote or Decrease the vote count of a Post

/delete_post/<post_id> Deletes the desired post
